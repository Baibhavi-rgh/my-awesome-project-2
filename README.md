AI Chatbot

The AI Chatbot for Digital Help is a web-based application designed to assist users in learning basic digital skills. It provides instant responses to user queries related to email usage, internet basics, cybersecurity, and general digital literacy.

**Objectives**
To improve digital literacy among users
To provide instant help for common digital queries
To demonstrate the use of AI in education

**Technologies Used**
Python
HTML, CSS (Frontend)
Basic NLP (Rule-based logic)
